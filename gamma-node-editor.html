        AI settings
        
      

      

        

          API keys are stored in this browser's localStorage in plain text and used only to call the chosen provider directly.
          They never leave your machine via this editor. Clear them any time with the button below.
        

        

          Local models run entirely in your browser via WebGPU. No data leaves your machine. The first time you Run, the model
          weights download (~1.5 GB for E4B, ~500 MB for E2B) and are cached in the browser; subsequent runs are offline. Requires a
          WebGPU-capable browser (Chrome, Edge, recent Safari) and a discrete GPU or unified-memory machine with at least 4 GB free.

        Edit key mapping
        
      

      

        

          Override individual key frequencies. Type a number (Hz) into any row to pin that key to a fixed pitch — overrides the default MIDI mapping and ignores octave shift while set. Leave blank to fall back to default. Stored locally in this browser.
        

        

        

          
          
        

      

    

  


  
  

    

      

        Edit slider curve
        
      

      

        

          Pick how the slider's drag position maps to its output. Linear is 1:1; log gives finer control at the bottom (good for frequency / amplitude); exp gives finer control at the top (good for time / decay); s-curve eases in and out.
        

        

      

    

  


  
  

    

      

        Edit envelope
        
      

      

        

          Click+drag to draw the envelope shape. The amber dashed line is the sustain marker — drag it left/right to choose where the curve holds while the gate is active. Indices left of the marker play during attack; the value at the marker is held; indices right play during release.

        Keyboard shortcuts & tips
        
      

      

        

          /Focus the palette filter
          ⌫Delete the selected node(s)
          ShiftHold while clicking nodes or marqueeing to add to selection
          EscClear selection / dismiss modal
          SpaceHold + drag to pan the canvas (or middle-click drag)
          WheelZoom toward cursor (Shift = finer step)
          0Reset view (zoom 100%, pan 0)
          ⌘ Z / ⌘ ⇧ ZUndo / redo
          ⌘ ASelect all nodes
          ⌘ SSave .gpatch
          1 / 2 / 3 / 4 / 5Switch tab (Properties / C++ / JSON / User DSP / Monitor)
          V / DSelect / Draw tool (pen, handwriting). Disabled while playing.
          Z / XOctave down / up while playing
          ⌘ ↵Run the AI prompt (when AI panel open)
        

        

          Wiring — drag a port onto another port to connect. Click an input port to pick up its existing wire and re-route. Drop on empty space to delete.
        

        

          Port types —  audio (solid),
           param (dashed),
           gate (trigger event),
           clock (sequencer rate).
